## Общая задумка
Пользователь создаёт анкету, где указывает кем работает или учится, хобби и увлечения, а также музыкальные предпочтения и загружает свою фотографию к себе в профиль. После регистрации и заполнения анкеты пользователь попадает в чат, где ему показываются различные анкеты других пользователей. Пользователь может поставить лайк другому пользователю и при взаимном лайке от другого пользователя происходит переход в личную переписку, где пользователи знакомятся и общаются. Нужна система модерации того, что происходит в приложении, как при регистрации и заполнении анкеты, так и в личной переписке. Приложение будет только десктоп под ОС Windows. Необходимо будет получать обратную связь от пользователей и совершенствовать продукт. Необходимо придумать уникальную фишку моего приложения.

## Стек технологий

- Языки и Фреймворки:
	- С++ - разработка desktop приложения. Для него будет использоваться framework QT, так как это наиболее популярный framework для построения графического интерфейса и со знанием QT можно легко найти работу.
	- Postgre SQL - для взаимодействия с бд. В бд будет хранится вся информация. Наиболее популярный диалект SQL, зная который также не сложно найти работу.
 - IDE:
	- Qt Creator
